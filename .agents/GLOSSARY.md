# Glossary

## Business Terms

POS

Point of Sale.

Customer

Business customer.

Sale

Completed transaction.

Held Sale

Temporarily stored cart.

Credit Sale

Customer pays later.

Payment

Debt collection.

Revenue

Total completed sales.

Inventory

Product stock.

SKU

Internal product code.

Barcode

Machine-readable product identifier.

---

## Technical Terms

SPA

Single Page Application.

PWA

Progressive Web Application.

CSR

Client Side Rendering.

Offline First

Application works without internet.

Firestore Cache

Persistent local database.

Snapshot Listener

Realtime Firestore subscription.

Batch Write

Atomic write operation.

Transaction

Multiple operations executed together.

Store

Global Zustand state.

Feature

Independent business module.

Barrel Export

index.ts public API.

Lazy Loading

Load code when needed.

Code Splitting

Separate bundles.

Hydration

React initialization process.
