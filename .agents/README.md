# Dijital Stok

Offline-first Inventory & POS application for SMEs.

Supports:

- Web
- PWA
- Android
- iOS

---

# Tech Stack

Frontend

- React 19
- TypeScript
- Vite

UI

- Tailwind CSS
- HeroUI

State

- Zustand

Backend

- Firebase Authentication
- Cloud Firestore

Native

- Capacitor

---

# Main Features

- Inventory Management
- Point of Sale
- Customer Management
- Credit Sales
- Analytics Dashboard
- Barcode Scanning
- Offline Support
- Multi Tenant
- Role Based Access Control

---

# Getting Started

Install

```bash
pnpm install
```

Run

```bash
pnpm dev
```

Build

```bash
pnpm build
```

Test

```bash
pnpm test
```

E2E

```bash
pnpm test:e2e
```
